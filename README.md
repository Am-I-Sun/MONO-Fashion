# MONO – Minimal Fashion Website

## Overview
MONO is a static fashion website designed with a minimal and modern aesthetic.  
The project showcases men’s and women’s clothing collections using clean layouts, neutral color palettes, and intuitive navigation.

## Objective
The objective of this project is to design a responsive and user-friendly fashion website using core web technologies while maintaining consistency and simplicity in design.

## Technologies Used
- HTML5
- CSS3
- VS Code
- Web Browser

## Project Structure
- `mono.html` – Home page
- `men.html` – Men’s collection page
- `women.html` – Women’s collection page
- `style.css` – Common stylesheet for all pages

## Features
- Responsive layout using CSS Grid and Flexbox
- Consistent navigation bar across pages
- Product display using card-based design
- Structured footer with brand information and links

## Responsiveness
The website adapts to different screen sizes using flexible layouts and media queries, ensuring usability across desktop and mobile devices.

## How to Run
Open `mono.html` in any modern web browser to view the website.
